# TI Space Invaders

I wrote this as a way to learn assembly for the calculator.
This has only been tested with the TI-84 Plus CE.

## Playing

### Controls

`dpad`: Left and right
`2nd`:  Fire
`clear`: Exit

## Build

To build the project, run the following command:

```shell
make
```

The game will be placed in the build folder.
